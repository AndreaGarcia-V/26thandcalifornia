We recommend that you start by reading the project overview on the Github project page.

Consult the Cook County Jail project's API Guide to learn about getting data out of the API.

Discalimer: the data on prisoners is known to be not accurate, both at the source, the Cook County Jail and the way the data is captured is not guaranteed to be fault free. Therefore do not take the data as definitive, rather, viewed it as suggestive.

Please note that the Stats views load the entire data set and the first time you request one of these views the loading of the data takes some time, upwards of a minute. Subsequent views should render much quicker.
